API文档

以下是Veomuse应用程序提供的API接口文档。

基础信息

Base URL: http://localhost:3000/api

认证: 所有API端点都需要有效的Gemini API密钥。您可以通过环境变量配置或在请求中提供API密钥。

模型接口

获取可用模型

Endpoint: GET /models

查询参数:

  • apiKey (可选): 您的Gemini API密钥

响应示例:

{
  "videoModels": [
    {
      "id": "veo-3.0-generate-preview",
      "name": "Veo 3.0 (Preview)"
    }
  ],
  "optimizationModels": [
    {
      "id": "gemini-2.5-pro",
      "name": "Gemini 2.5 Pro"
    }
  ]
}

文字生成视频

生成视频

Endpoint: POST /text-to-video

请求体:

{
  "text": "A beautiful sunset over the ocean",
  "negativePrompt": "blurry, low quality",
  "apiKey": "your-api-key",
  "model": "veo-3.0-generate-preview",
  "optimize": true,
  "socketId": "socket-id-for-realtime-updates"
}

响应示例:

{
  "success": true,
  "message": "Video generation started",
  "operationName": "operations/...",
  "usedApiKey": "used-api-key",
  "usedModel": "veo-3.0-generate-preview"
}

图片生成视频

生成视频

Endpoint: POST /image-to-video

表单数据:

  • image: 要上传的图片文件
  • prompt: 如何生成视频的描述
  • negativePrompt: 要避免的元素
  • apiKey: 您的API密钥
  • model: 使用的模型
  • optimize: 是否优化提示词
  • socketId: 实时更新的Socket ID

响应示例:

{
  "success": true,
  "message": "Video generation started",
  "operationName": "operations/...",
  "usedApiKey": "used-api-key",
  "usedModel": "veo-3.0-generate-preview"
}

提示词优化

优化提示词

Endpoint: POST /optimize-prompt

请求体:

{
  "prompt": "A cat playing",
  "apiKey": "your-api-key",
  "model": "gemini-2.5-pro"
}

响应示例:

{
  "success": true,
  "optimizedPrompt": "A playful orange cat chasing a ball of yarn in a sunlit living room"
}

操作状态

检查操作状态

Endpoint: GET /operation/:operationName

查询参数:

  • apiKey: 您的API密钥
  • socketId: 实时更新的Socket ID

响应示例:

{
  "done": false,
  "response": {
    "generateVideoResponse": {
      "generatedSamples": [
        {
          "video": {
            "uri": "https://..."
          }
        }
      ]
    }
  }
}

视频管理

下载生成的视频

Endpoint: POST /download-video

请求体:

{
  "videoUri": "https://...",
  "apiKey": "your-api-key"
}

响应示例:

{
  "success": true,
  "message": "Video downloaded successfully",
  "videoPath": "generated/video-12345.mp4"
}

转码视频

Endpoint: POST /transcode-video

请求体:

{
  "inputPath": "generated/video-12345.mp4",
  "format": "webm",
  "resolution": "720p",
  "fps": 30,
  "socketId": "socket-id-for-realtime-updates"
}

响应示例:

{
  "success": true,
  "message": "Video transcoded successfully",
  "videoPath": "generated/transcoded-12345_720p_30fps.webm"
}

文件访问

生成的视频: /generated/:filename

上传的图片: /uploads/:filename

健康检查

Endpoint: GET /health

响应示例:

{
  "status": "ok",
  "timestamp": "2023-01-01T00:00:00.000Z",
  "gpuSupport": {
    "nvidia": true,
    "intel": true,
    "amd": false
  }
}
