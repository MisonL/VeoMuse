## 重点差距（按影响优先）

1. 权限隔离：工作区渠道接口需从组织角色切换为工作区角色。
2. 数据完整性：评论分页游标需升级为稳定复合游标。
3. 操作正确性：策略创建防重提交必须补齐。
4. 可访问性：渠道弹窗与模式切换的键盘/语义支持需达标。
5. 门禁可信度：统一 CI 实例与回归覆盖范围，消除假绿。

## 建议落地顺序

1. P0/P1 安全与正确性（权限、分页、防重、弹窗键盘）。
2. S0 门禁可信度（端口统一、默认全量回归）。
3. P2 体验与可访问性（语义状态、空态/失败态、FPS 文本替代）。
4. 架构治理（大文件拆分、类型收敛）。
