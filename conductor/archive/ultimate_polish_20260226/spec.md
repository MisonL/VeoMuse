# Specification: VeoMuse 旗舰版终极性能与架构打磨

## 1. 业务目标
将 VeoMuse 从“功能完整”提升至“工业级完美”，建立可量化的性能指标，实现架构的深度解耦与极致的 UI/UX 体验。

## 2. 功能需求
- **[后端] 架构大一统**: 强制所有 AI 业务服务继承自 `BaseAiService`，实现全站统一的重试、日志与耗时量化。
- **[后端] 清理样板代码**: 移除路由层冗余的 try-catch，利用 Elysia 全局拦截器实现错误闭环。
- **[性能] 预览引擎 Native 化**: 剥离预览同步逻辑，通过独立的同步控制器直接操作视频 Ref，极大降低 React 调度开销。
- **[UI/UX] 现代化美学重塑**:
    - 全站统一使用玻璃拟态 Toast 反馈。
    - 调优入场动画的物理弹性曲线（Spring Physics）。
    - 对侧边栏和控制台执行“现代化美学”风格化处理。
- **[安全] Nginx 堡垒加固**: 补齐 CSP 安全响应头，防止脚本注入。
- **[维护] 自动资源回收**: 实现 Cron 任务定期清理失败的合成产物与临时文件。

## 3. 验收标准
- 控制台可清晰量化每一路 AI 调用的耗时与重试次数。
- 120s 时间轴在播放时无卡顿（FPS 保持在 55+）。
- 模拟断网或 API 失败时，UI 提供完美的降级与自愈提示。
