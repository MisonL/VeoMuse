## 🌟 项目定义：AI 创作的巅峰形态

**VeoMuse V3.1 Pro** 是一款专为 2026 年 AI 创作生态打造的旗舰级应用。它打破了 2D 视频生成的局限，通过**全球模型总线**集成了 Sora、Kling、Luma 等顶级驱动，并配合**神经动捕**与**3D 空间感知**，实现了从创意灵魂到沉浸式空间视频的一键式闭环。

## ✨ 核心功能矩阵

### 1. 🎬 工业级编辑器 (Pro Editor)
- **多维轨道系统**：支持视频、音频、文字、蒙版（Mask）四类轨道并行编排。
- **卓越交互**：支持**磁吸对齐**、**1x-100x 动态缩放**、**无限级撤销 (Undo/Redo)**。
- **实时同步**：基于 RAF (requestAnimationFrame) 的高精度音画文同步预览。

### 2. 🤖 全自动化导演 (Auto-Director)
- **一键成片**：输入长脚本，AI 自动完成：**拆解分镜 -> 指挥生成 -> 匹配 BGM -> 合成配音 -> 自动排版**。
- **World-Link**：跨镜头全场景一致性锁定技术。

### 3. 🧠 全球模型总线 (Model Bus)
- **顶级渠道**：集成 Gemini 3.1, Sora, Kling, Luma Dream Machine, Runway Gen-3, Pika 1.5。
- **对比实验室**：分屏对比视图，实时 PK 不同模型的生成效能。

### 4. 🧬 媒体炼金术 (Alchemy)
- **语种重塑**：多语种配音同步翻译并自动对口型 (Lip-Sync)。
- **视觉迁移**：一键将视频重构为“梵高”、“赛博朋克”或“吉卜力”风格。
- **空间升维**：利用 NeRF 技术将 2D 视频升维为 3D 空间场景。

## 🛠️ 技术栈 (State-of-the-art)

- **后端**：Bun + ElysiaJS + BaseAiService (抽象基类) + Eden Treaty。
- **前端**：React 19 + Vite 8 (Rolldown) + Framer Motion (Spring Physics)。
- **性能**：全链路 API 响应量化监控，FPS 级渲染优化。
- **持久化**：PostgreSQL + Redis + Docker + Nginx。
