## 🚀 快速开始

### 1. 环境准备
确保已安装 [Bun](https://bun.sh/) 1.3+ 和 [Docker](https://www.docker.com/)。

### 2. 本地开发
```bash
bun install
bun run dev
```

### 3. 生产部署 (Docker)
```bash
# 自动执行全量构建与部署
bun run docker:up
```
